Remove Characters

Type: Remove Rename

The Remove Characters function has two modes:
  • In Range - Allows the selction of a range of characters to remove from each filename.  If the beginning of the range is longer than the filename in a given row, that row will not be renamed.  If the beginning of the range is within a filename and the the end of the range exceeds the length of a filename, that file will be renamed.  To avoid changing file extensions, the Hide Extensions option can be used.
  • From End - The number of characters specified will be removed from the end of each filename
