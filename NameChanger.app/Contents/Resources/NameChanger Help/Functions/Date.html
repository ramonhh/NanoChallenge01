Date

Type: Date Rename

Date renaming allows the addition of a date string to each filename.  The details controlling the Date rename function are presented in a separate dialog that appears when Date is selected as the rename type.


The Date rename function has three modes which can be selected to the right of the rename type combo box:
  • Custom Date - Custom date allows the user to specify a date from the DatePicker in the Date dialog.
  • File Date - Take the file modification date for each file.  The value in the DatePicker is ignored.
  • EXIF Date - Take the date from the EXIF information (if present) for each file.  The value in the DatePicker is ignored.
All modes allow the selection of format from one of 6 possibilities, and the location of the date.  Append will place the date string at the end of the filename.  To keep the original extensions enable the Hide Extensions option.  Prepend will place the date string at the beginning of the filename.
