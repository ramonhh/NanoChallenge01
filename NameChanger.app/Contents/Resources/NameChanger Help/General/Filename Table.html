Filename Table

The filename table is a listing of all the files that have are currenlty in NameChanger.  It also has a number of features designed to make renaming files easier.

Table Features

  • Checkboxes - The checkbox column on the left edge of the table lets the user skip certain files during the rename process.  A row with a checked checkbox indicates that it should undergo renaming, which a row with an empty checkbox will be skipped.
  • Selection - Selection of rows can be used to either remove or rearrange items in the table.  Dragging a selection up or down will move those rows together into the location they are dropped at.  Clicking Remove will remove any selected rows.
  • Reveal in Finder - Right clicking on an file in the Original Filename column will reveal a context menu containing a Reveal in Finder option that will open a Finder window to the location of the chosen file.
  • Grey Rows - A grey row indicates that there is an error that will prevent that row from being renamed.  The normal causes for this are that a file with the specified name already exists or the file to rename has invalide permissions, been moved or deleted.   Mousing over the row will reveal a tooltip that describes which error has been encountered.  Choosing to rename while there are grey rows will rename the files that can be changed, any that cannot will be skipped.
