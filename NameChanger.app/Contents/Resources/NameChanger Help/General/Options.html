Options

The Options dialog is presented by clicking the Options toolbar item.



The Options menu houses a number of important features to make renaming files easier.

General Options

  • Hide Extensions - Enabling this checkbox will hide the extension in the Original Filename and Renamed Filename columns.  This allows for the changing of the filename without affecting the extension of the resulting renamed file.  Useful for rename functions such as Append, Remove Characters, Sequence (append and replace entire filename) and Date (append).  By default this is not check, however there is a preference which sets it to checked by default.
  • Ignore Case - Ignores any case differences between the Original Text field and the filename presented in the Original Filename column.  This only applies to the following rename functions: Replace First Occurrence, Replace Last Occurrence, Replace All Occurrences and Wildcard.
  • Force Unique - If the results of the rename will more than one file to be renamed to the same name, this option will add a sequence number (starting at 1) to the end of the filename, but before the extension.  This is only applicable for the Remove Characters and Wildcard functions.

Checked Options

  • Check All - Checks all the files in the table.
  • Uncheck All  - Unchecks all the files in the table
  • Invert Checks - All checkboxes that were checked will now be unchecked and all that were unchecked will now be checked.

Selection Options

  • Select All - Selects all the files in the table.
  • Deselect All  - Deselects all the files in the table
  • Invert  File Selection - All files that were selected will now be deselected and all that were deselected will now be selected.
