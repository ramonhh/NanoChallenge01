Image Browser

The Image Browser allows a visual inspection and reordering of the images being renamed.


Overview

To display the Image Browser, click the Image Browser button in the toolbar.  The Image Browser appears as a separate dialog where images will be shown as a thumbnail and non-images are represented by their icon.  Each item in the Image Browser has two lines of text below.  The first line(bold text) is mapped to the contents of the Renamed Filename column.  The second line(grey text) is mapped to the contents of the Original Filename column.
Right clicking on an item in the Image Browser will present a context menu with a Reveal in Finder option.  This will open a Finder window to the location of the selected item.


Sequence Renaming

Items within the Image Browser can be dragged to other locations, allowing for a visual re-ordering of images.  This is especially useful for sequence renaming when the order of the pictures is important.
