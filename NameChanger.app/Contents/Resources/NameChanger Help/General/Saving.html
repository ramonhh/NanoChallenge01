Saving

Saving allows for frequently used renaming operations to be stored for later use.  

Saving a Rename

  1. Enter desired information into the current rename.
  2. From the Renames menu select Save Current Rename...
  3. Enter a title for the current rename, this is the name that will be shown in the menu.
  4. If a duplicate name is chosen, the user is presented with option of replacing the previous version.

Loading a Rename

  1. Open the Renames menu to view saved renames
  2. Select the desired rename

Managing Renames

  • Select Manager Renames... from the Renames menu
  • The Rename Management sheet shows the title and type of each rename
  • The Management sheet allows for the following:
    • Removal of a save renames by selecting one or more renames and click the minus button
    • Changing the title of the saved rename by double clicking the title
